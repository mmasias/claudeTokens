Configuración de la extensión

Native Messaging Host

La extensión lee los archivos de sesión de ~/.claude/projects/ mediante un script Python local. Instálalo una vez por máquina ejecutando:

cd native_host
chmod +x install.sh
./install.sh TU_EXTENSION_ID

El ID de la extensión está en chrome://extensions (modo desarrollador activado).

Límite semanal (opcional)
